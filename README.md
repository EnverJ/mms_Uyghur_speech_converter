# MMS Uyghur Speech Converter

A simple Python project that records speech from a microphone, transcribes it using **Meta MMS (Massively Multilingual Speech)**, and converts Uyghur Latin transcription into **Uyghur Arabic script**.

This project is designed for learning, experimentation, and building Uyghur speech datasets.
